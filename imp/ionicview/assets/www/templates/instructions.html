Let me help you get started...

First Step
Install Node.js. Then, install the latest Cordova and Ionic command-line tools.
$ npm install -g cordova ionic
Second Step
Using the Ionic CLI, create an Ionic project using one of our ready-made app templates.
$ ionic start myApp sidemenu
Third Step
Upload your app! It will now appear in your list of apps in Ionic View.
$ cd myApp && ionic upload
